Welcome to CathyJS
=================

CathyJS is an api wrapper for powerful chatbot API.


Usage
------------

### ← Installing

```js
npm i cathyjs
```

### ← Example

```js
const cathyjs = require("cathyjs");

let res = cathyjs.startChatting("hello"); // replace your message

console.log(res);
// res will be reply message. 
```
